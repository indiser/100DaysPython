# COURSE SYLLABUS: 100 Days of Code - The Complete Python Pro Bootcamp

**Instructor:** Lead Instructor, The App Brewery (London)  
**Duration:** 100 Days (Approx. 60+ Hours of HD Video Content)  
**Pacing:** 1 Hour Per Day / 1 Project Per Day  
**Format:** Self-paced online lectures, hands-on projects, and code challenges  

---

## 1. Course Overview
This comprehensive bootcamp is designed to take students from absolute zero programming experience to professional Python developer in 100 days. Through a project-based approach, students will build 100 real-world applications, learning the latest tools and technologies utilized by top-tier companies like Apple, Google, and Netflix. 

## 2. Prerequisites & Requirements
* **Experience:** None required. Suitable for complete beginners to intermediate programmers.
* **Hardware:** A 64-bit Mac or PC with at least 4GB of RAM.
* **Internet:** Reliable internet connection required.
* **Software:** No paid software is required. Step-by-step setup is provided for PyCharm, Jupyter Notebooks, and Google Colab.

## 3. Target Audience
* **Beginners:** Individuals with zero coding experience wanting a complete path to becoming a Python professional.
* **Entrepreneurs:** Aspiring founders wanting to build their own websites, web apps, and startups.
* **Career Switchers:** Seasoned programmers looking for the quickest, project-based route to transition to Python.
* **Intermediate Coders:** Existing Python developers looking to level up their skills through 100 days of rigorous challenges.

---

## 4. Comprehensive Curriculum Modules
*Note: Technologies are taught progressively over the 100-day period.*

### Module 1: Python Fundamentals & Environment Setup
* **Core Language:** Python 3 (Latest Version)
* **Development Environments:** PyCharm, Jupyter Notebook, Google Colab
* **Version Control:** Bash Command Line, Git, and GitHub

### Module 2: Graphical User Interfaces (GUI) & Game Development
* **Libraries:** Turtle, Tkinter (Desktop App Development)
* **Concepts:** Object-oriented programming, game loops, state management, event-driven programming.

### Module 3: Automation, Web Scraping & APIs
* **Web Scraping Libraries:** Beautiful Soup, Selenium Web Driver
* **API Integrations:** Requests module, building RESTful APIs, handling Authentication
* **Automation:** Scripting routine tasks, browser automation, data extraction

### Module 4: Full-Stack Web Development
* **Front-End Technologies:** HTML 5, CSS 3, Bootstrap 4
* **Back-End Frameworks:** Flask, WTForms
* **Databases:** SQL, SQLite, PostgreSQL
* **Deployment:** GitHub Pages, Heroku, GUnicorn

### Module 5: Data Science & Analysis
* **Data Manipulation & Math:** Pandas, NumPy
* **Data Visualization:** Matplotlib, Plotly, Seaborn
* **Machine Learning Fundamentals:** Scikit-learn
